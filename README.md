# canvas 学习练习

canvas 知识学习

## 快速开始

### 运行

输入命令

```bash
  npm run dev
```

### 重新生成目录

```bash
  npm run compile
```

执行命令会在`template`目录下生成`menu.json`的文件,`index.js`会自动读取`menu.json`中的内容并生成菜单

访问地址 `localhost:8081`
